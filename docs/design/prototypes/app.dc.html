Тесты почти зелёные, но два падают. Посмотрите на кейс со 100 000 событий: что происходит со временем работы вашего решения при росте входа?
Ответить голосом или текстом…
Задача: дедупликация событий solution.js 
function dedupe(events) {
  const seen = [];
  return events.filter((e) => {
    if (seen.includes(e.id)) return false;
    seen.push(e.id);
    return true;
  });
}

Правовая информация

Обновлено 1 июля 2026 · действует для сервиса «Хекслет Карьера»

1. Общие положения

Сервис помогает в поиске работы: хранит резюме и письма, агрегирует вакансии из открытых источников, ведёт учёт откликов и проводит тренировочные интервью с AI. Используя сервис, вы соглашаетесь с этими условиями и Пользовательским соглашением Хекслета. Сервис предназначен для лиц старше 16 лет.

2. Персональные данные

Данные резюме и профиля обрабатываются по 152-ФЗ. Резюме не передаются работодателям без вашего явного действия. Записи тренировок хранятся 30 дней и видны только вам. Выгрузка или удаление данных — в настройках или через privacy@hexlet.io.

3. AI-функции

Ревью резюме, генерация писем и тренировочные интервью используют большие языковые модели — их ответы носят рекомендательный характер. Код в тренировках проверяется автотестами в изолированной среде; решения не используются для обучения моделей без отдельного согласия.

Важно: сервис создан для тренировки. AI-подсказки во время реальных собеседований могут нарушать правила компаний и привести к отзыву оффера — мы сознательно не делаем таких функций.

4. Подписка и ответственность

Базовые функции бесплатны; условия тарифа «Про» — на странице тарифов, отмена — в любой момент. Сервис агрегирует вакансии «как есть», не является кадровым агентством и не гарантирует трудоустройство. Товарные знаки компаний принадлежат их правообладателям.
